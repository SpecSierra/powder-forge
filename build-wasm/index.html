Powder Forge
Physics Sandbox

Powder Forge is a modernised interface for The Powder Toy, the classic open-source physics sandbox game. Everything you love about the original — every element, every reaction, every saved simulation — is still here.

This version adds a refreshed dark UI, TrueType font rendering, and runs entirely in your browser via WebAssembly. No installation required.

All game logic, saves, and online features belong to powdertoy.co.uk. Powder Forge is an independent UI project and is not affiliated with or endorsed by the original Powder Toy team.
